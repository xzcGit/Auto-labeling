# 这是传入的数据集文件夹结构
```
├── root_path
│   ├── category1
|   |   ├── images
│   │   |   ├── 1.jpg
│   │   |   ├── 2.jpg
│   │   |   ├── 3.jpg
│   │   |   ├── 4.jpg
│   │   |   ├── 5.jpg
|   |   ├── labels
|   |   ├── pre_images
│   │   |   ├── 1.jpg
│   │   |   ├── 2.jpg
│   │   |   ├── 3.jpg
|   |   ├── pre_labels
│   │   |   ├── 1.txt
│   │   |   ├── 2.txt
│   │   |   ├── 3.txt
│   ├── category2
|   |   ├── images
│   │   |   ├── 1.jpg
│   │   |   ├── 2.jpg
│   │   |   ├── 3.jpg
│   │   |   ├── 4.jpg
│   │   |   ├── 5.jpg
|   |   ├── labels
|   |   ├── pre_images
│   │   |   ├── 1.jpg
│   │   |   ├── 2.jpg
│   │   |   ├── 3.jpg
|   |   ├── pre_labels
│   │   |   ├── 1.txt
│   │   |   ├── 2.txt
│   │   |   ├── 3.txt

```
# 其中，root_path是传入的文件夹路径，category1和category2是类别文件夹，images是待标注的图像，labels用于存放待标注图像的标签，pre_images中是已标注的图像，pre_labels是已标注图像的标签。
